
# MERN Typescript docker Boilerplate

This is a Boilerplate template for a MERN typescript app with docker to run everything

## How to run
``docker-compose up --build``

Structure

Backend
```
├── package-lock.json
├── package.json
├── src
│   ├── app.ts
│   ├── controllers
│   ├── middleware
│   ├── models
│   │   └── user.ts
│   ├── routes
│   │   └── user.ts
│   ├── server.ts
│   └── services
└── tsconfig.json
```

Frontend
```
├── package-lock.json
├── package.json
├── public
├── src
│   ├── App.test.tsx
│   ├── App.tsx
│   ├── actions
│   ├── components
│   ├── index.tsx
│   ├── pages
│   ├── react-app-env.d.ts
│   ├── reportWebVitals.ts
│   ├── setupTests.ts
│   ├── store
│   └── utils
└── tsconfig.json
```
